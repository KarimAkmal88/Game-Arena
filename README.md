## 🛠️ Tech Stack & Tools Used

- **HTML5:** Semantic architecture (`<header>`, `<main>`, `<section>`, `<footer>`) for better SEO and structural clarity.
- **Bootstrap v5.3.8:** Integrated the official latest framework for seamless layout grids and modern responsive controls.
- **Custom CSS3:** Engineered highly optimized performance logic, eliminating unused Tailwind styling to maintain lightweight code.
- **Font Awesome v6.5.1:** Loaded the latest stable iconography for pixel-perfect social and technical branding (including updated X/Twitter and Discord icons).

---

## ✨ Features & Interactive UI

- **Responsive Grid System:** Pixel-perfect adaptive scaling tailored specifically for high-density smartphone testing and large displays.
- **Tactical Team Cards:** Powered by `backdrop-filter: grayscale()` to produce a premium, fluid transition effect inspired by AAA competitive gaming profiles (VALORANT / Cyberpunk style).
- **Marquee Carousel System:** Tailored dual-row loop animations engineered to run via native GPU acceleration for locked 60+ FPS performance.
- **Premium Button Shimmer:** Integrated advanced CSS pseudo-elements (`::after`) to execute real-time kinetic shine effects during cursor hover.
- **Secure Input Ringing:** Adapted Bootstrap form focus loops to fire a neon high-gloss primary glow upon interaction.
